Chapter 2. Basic Facts about Natural Deduction

In the previous chapter, we introduced the proof theory for Pred1, where we adopt the natural deduction style proof theory for first order logic. In defining the proof theory, we first gave the inductive definition of the provable relation, which is a collection of 18 inference rules (two of which could be called axioms since they do not have any hypothesis). We also defined that, formally, a proof for a judgment is a derivation tree, of which each node is an instance of one such inference rules and the root of the tree being the judgment in question. But in practice, it soon becomes very repetitive in writing proof trees if we restrict ourselves to the 18 inference rules given in the definition of provable relation. For example, we already proved the commutativity of $\wedge$ in exercise 3.1.6 but if we restrict ourselves to the 18 inference rules, we have to write a proof tree with a size of 6 to prove $\phi\wedge\psi\judges\psi\wedge\phi\vee\gamma$, but we know that metalogically, we can abbreviate the part of the proof concerns the commutativity of $\wedge$ into one step since we already have the result of exercise 3.1.6. In abbreviating those steps, we essentially treated the result of exercise 3.1.6 as a new inference rule, which is acceptable since given a well formed abbreviated proof tree, we essentially demonstrated that metalogically we can construct a proof tree using only the definitive reference rules. The goal of this chapter, is then to explore and prove several commonly used results about the natural deduction system that can be used in a similar way as the result of commutativity of conjunction mentioned above.

Facts about Logical Conjunction & Disjunction

Prove the following facts.

  1. reflective law: $\infr{\quad \Gamma\judges\phi\quad}{\Gamma\judges\phi\wedge\phi}$, $\infr{\quad \Gamma\judges\phi \quad}{\Gamma\judges\phi\vee\phi}$

  2. commutative law: $\infr{\Gamma\judges\phi\wedge\psi}{\Gamma\judges\psi\wedge\phi}$, $\infr{\Gamma\judges\phi\vee\psi}{\Gamma\judges\psi\vee\phi}$

  3. distributive law: $\infr{\quad \Gamma\judges\gamma\wedge(\phi\vee\psi) \quad% }{\Gamma\judges(\gamma\wedge\phi)\vee(\gamma\wedge\psi)}$, $\infr{\quad \Gamma\judges\gamma\vee(\phi\wedge\psi) \quad% }{\Gamma\judges(\gamma\vee\phi)\wedge(\gamma\vee\psi)}$

  4. De Morgan’s law: $\infr{\quad \Gamma\judges\neg(\phi\wedge\psi) \quad}{\Gamma\judges(\neg\phi)\vee(\neg\psi)}$, $\infr{\quad \Gamma\judges\neg(\phi\vee\psi) \quad}{\Gamma\judges(\neg\phi)\wedge(\neg\psi)}$

Facts about Logical Entailment

Prove the following facts.

  1. paraphrase: $\infr{\Gamma\judges\phi\rightarrow\psi}{\Gamma\judges\neg\phi\vee\psi}$, $\infr{\Gamma\judges\neg\phi\vee\psi}{\Gamma\judges\phi\rightarrow\psi}$

  2. contraposition: $\infr{\quad \Gamma\judges\phi\rightarrow\psi \quad}{\Gamma\judges\neg\psi\rightarrow\neg\phi}$

Facts about Quantifiers

Prove the following facts.

  1. De Morgan’s law: $\infr{\quad \Gamma\judges\forall{x}\phi \quad}{\Gamma\judges\neg\exists{x}\neg\phi}$, $\infr{\quad \Gamma\judges\exists{x}\phi \quad}{\Gamma\judges\neg\forall{x}\neg\phi}$, $\infr{\Gamma\judges\neg\forall{x}\phi}{\Gamma\judges\exists{x}\neg\phi}$, $\infr{\Gamma\judges\neg\exists{x}\phi}{\Gamma\judges\forall{x}\neg\phi}$

  2. generalization rule: $\infr{\Gamma\judges\phi \hyposp x\not\in fv(\Gamma)}{\Gamma\judges\forall{x}\phi}$

  3. $\alpha$-equivalence: $\infr{\Gamma\judges\forall{x}\phi \hyposp y\not\in fv(\phi)}{\Gamma\judges\forall{y}(\phi[x\mapsto y])}$ $\infr{\Gamma\judges\exists{x}\phi \hyposp y\not\in fv(\phi)}{\Gamma\judges\exists{y}(\phi[x\mapsto y])}$

Misc

Prove the following facts.

  1. Excluded Middle: $\infr{\qqquad\qquad}{\Gamma\judges\phi\vee\neg\phi}$

  2. Peirce’s law: $\infr{\qqquad\qquad\qqquad\qqquad% }{\Gamma\judges((\phi\rightarrow\psi)\rightarrow\phi)\rightarrow\phi}$

  3. Equality Theorem: $\infr{\Gamma\judges\phi[x _1\mapsto s _1,\dots,x _k\mapsto s _k] \hyposp % s _1 = t _1 \;\; \dots \;\; s _k = t _k % }{\Gamma\judges\phi[x _1\mapsto t _1,\dots,x _k\mapsto t _k]}$
